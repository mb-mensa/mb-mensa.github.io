# [mb-mensa.github.io](https://mb-mensa.github.io)

HTML viewer for the Migros Bank Personalrestaurant menu plan publicly available at
[betriebsrestaurants-migros.ch/landingpages/migrosbank/info-menueplan](https://www.betriebsrestaurants-migros.ch/landingpages/migrosbank/info-menueplan). This page simply parses the PDF to HTML once a week to save you some clicks and bandwidth.

Inspired by [mensa.davidemarcoli.dev](https://mensa.davidemarcoli.dev), but much simpler and with fewer features. Hosted via [GitHub Pages](https://docs.github.com/en/pages) from this repo.
